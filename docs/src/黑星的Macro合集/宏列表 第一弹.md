## 1. 随机宏
> [!INFO]
> 链接: https://github.com/Raven-Book/TweeGoGuide/tree/master/src/macro/prize.min.js
> * **基本使用** <span style="color: green;">**已测试**</span>
> * **嵌套使用** <span style="color: red;">**未测试**</span>

`<<prize 选项与其概率>> <</prize>>`

随机出的选项可以通过 `_result` 或 `$result` 在宏内部使用，注意这两个变量只在宏内部有效。

```SugarCube
<<prize `{"厕所": 0.9, "浴室": 0.1}`>>
<<print _result>>
<<print $result>>
<</prize>>
```
